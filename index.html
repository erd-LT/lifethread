⚡ Anthropic API Key

How to get your free API key:
  1. Go to platform.anthropic.com
  2. Sign up or log in
  3. Click API Keys in the left menu
  4. Click Create Key — copy the key
  5. Paste it below and tap Save
The key is stored only on this device. It never leaves your browser.
